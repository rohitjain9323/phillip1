## ğŸ› Troubleshooting

### Problem: "Font not found" error

**Solution:**
- Make sure you installed **all weights** of Inter and Outfit fonts
- Required weights: Light, Regular, Medium, SemiBold, Bold
- **Restart Figma** after installing fonts

### Problem: Plugin doesn't appear in menu

**Solution:**
- Make sure you ran `npm run build` successfully
- Check that `code.js` file exists in the `figma-plugin` folder
- Try restarting Figma
- Re-import the plugin: Menu â†’ Plugins â†’ Development â†’ Import plugin from manifest...

### Problem: "Cannot find module" error

**Solution:**
```bash
# Delete node_modules and reinstall
rm -rf node_modules
npm install
npm run build
```

### Problem: Plugin crashes or shows errors

**Solution:**
- Open Figma Console: Menu â†’ Plugins â†’ Development â†’ **Open Console**
- Check for error messages
- Make sure all fonts are installed
- Try running on a new/empty Figma file

### Problem: Screens look different than expected

**Solution:**
- Verify you have all font weights installed
- Check that fonts are named exactly "Inter" and "Outfit"
- Some font managers may rename fonts - use the system font installer
